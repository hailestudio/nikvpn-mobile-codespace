## 📋 نسخه 2.0 (نسخه Mobile)

### ✨ ویژگی‌های جدید

- **دو کانفیگ یکجا**: VLESS xHTTP و VLESS XTLS
- **QR Codes خودکار**: تولید و نمایش QR Code برای هر کانفیگ
- **منوی تعاملی**: کنترل کامل از ترمینال
- **پشتیبانی موبایل**: طراحی شده برای استفاده از موبایل
- **بهینه‌سازی عملکردی**: کانفیگ XTLS با بازدهی بالاتر
- **DNS بهبود‌یافته**: تحل‌کننده DNS تکراری برای مناطق مختلف
- **Keepalive خودکار**: اتصال مداوم در پس‌زمینه
- **لاگ بهتر**: پیام‌های فارسی و واضح

### 🔄 بهبود‌های موجود

- تنظیمات Xray برای موبایل بهینه‌شده
- پیام‌های مزید به فارسی
- رابط کاربری بهتر و واضح‌تر
- اسکریپت‌های بهتر شده و سازمان‌یافته
- خطاهای بیشتر پوشش‌داده شده

### 🐛 رفع خطاها

- مشکل UUID چند‌بار تولید شدن
- مشکل SNI در کانفیگ‌های پویا
- مشکل پورت‌های انتشار نشده
- بهبود مدیریت نشست‌های tmux

## 📝 یادداشت‌های نسخه

### 2.0.0 - 2024

**راه‌اندازی:**
```bash
git clone https://github.com/nikvpn-iran/nikvpn-mobile-codespace.git
cd nikvpn-mobile-codespace
gh codespace create --repo nikvpn-iran/nikvpn-mobile-codespace
```

- بازنویسی کامل ساختار
